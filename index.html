Projects

Multilingual Video Translation System

Built an end-to-end AI pipeline that converts spoken content in videos into translated audio in another language. The system integrates speech-to-text using Whisper, translation, and text-to-speech synthesis while maintaining audio-video synchronization using FFmpeg.

Chemical Safety RAG Assistant

Developed an offline Retrieval-Augmented Generation system that answers natural language queries over Chemical Safety Data Sheets. Implemented vector search using FAISS and MiniLM embeddings with LangChain and local LLMs via Ollama.

Adaptive Diagnostic Engine

Designed an adaptive testing backend that dynamically selects questions using an ability-estimation algorithm inspired by Item Response Theory. Built REST APIs using FastAPI and MongoDB to manage test sessions and generate personalized study plans with LLMs.
